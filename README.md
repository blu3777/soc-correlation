# SOC Correlation Engine

Projeto em Python que simula um mecanismo de correlação de eventos de segurança, aproximando-se da lógica utilizada em ambientes de SOC.

## Objetivo
Demonstrar como múltiplos eventos isolados podem ser correlacionados para identificar incidentes de segurança mais complexos.

## Funcionalidades
- Leitura de eventos de segurança simulados
- Correlação entre diferentes tipos de eventos
- Identificação de possíveis incidentes
- Geração de alertas correlacionados
- Simulação de triagem em SOC

## Tecnologias Utilizadas
- Python 3
- Análise de eventos
- Correlação baseada em regras
- Estruturas de dados

## Como Usar
1. Clone o repositório
2. Execute o script principal
3. O sistema processará os eventos configurados
4. Alertas correlacionados serão exibidos no terminal

## Conceitos de Segurança Aplicados
- SOC (Security Operations Center)
- Correlação de eventos
- Incident Detection
- Threat Analysis

## Observação
Projeto educacional com foco em aprendizado e demonstração de conceitos de SOC.
